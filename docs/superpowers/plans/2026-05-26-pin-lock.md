# PIN Lock Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a local PIN lock so the owner can keep the installed YOLK app private on laptop and iPhone.

**Architecture:** Keep business records and Supabase payload unchanged. Store local-only PIN lock settings in `egg_pin_lock`, derive lock/unlock state inside the existing Alpine app, and render an unlock overlay plus Cloud Sync management panel.

**Tech Stack:** HTML, Tailwind, Alpine.js, Lucide icons, browser localStorage, Node test runner.
